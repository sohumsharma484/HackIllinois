# Anchor Solana Program

```shell
anchor build
anchor deploy
```

Copy the **program ID** from the output logs; paste it in `Anchor.toml` & `lib.rs`.

```shell
anchor build
anchor deploy

yarn install
yarn add ts-mocha

anchor run test
```

generate c# client: 
dotnet anchorgen -i target/idl/lumberjack.json -o target/idl/ProgramCode.cs
